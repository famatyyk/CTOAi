# Run 001 - Azure Activity Agent Eval

Inputs:
- dataset.jsonl
- prompt variants in ../../prompt-variants/

Outputs:
- results.template.jsonl (copy to results.<variant>.<model>.jsonl and fill)
- results.<variant>.<model>.jsonl.summary.json (generated by aggregator)

Execution steps:
1. Choose variant prompt file.
2. Run each case from dataset.jsonl through chosen model/prompt.
3. Record one JSON object per case in results file.
4. Aggregate metrics:
- python scripts/ops/aggregate_agent_eval.py evals/runs/run-001/results.<variant>.<model>.jsonl
5. Compare summary files across variants; promote only if metrics improve without safety regressions.
